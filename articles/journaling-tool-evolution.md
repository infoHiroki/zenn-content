# Notion → Obsidian → Logseq → 結局これ

## 😩 俺は日記が続かない

日記。ジャーナリング。振り返り。

呼び方はなんでもいい。要するに「今日何やったか書く」ってやつだ。

😇 続かない。マジで続かない。

🫠 Notion使ってみた。Obsidian使ってみた。Logseq使ってみた。全部ダメだった。いや、ツールは悪くない。俺が悪い。俺の意志が豆腐より柔らかいのが悪い。

でも今は続いてる。**GitHub + Claude Code + Googleカレンダー**という謎の組み合わせで。

✨ なぜ続いてるのか。簡単だ。**書いてないから**だ。

---

## 🗺️ 変遷を振り返る（振り返りツールの話なのに）

```
Notion → Obsidian + Claude Desktop → Logseq → GitHub + Claude Code + Googleカレンダー
```

見事な放浪っぷりだ。デジタルノマドならぬデジタル浮浪者。

🤖 でもこの放浪には一応の軸がある。**AIとの相性**だ。

---

## 1️⃣ Notion時代 - データベース地獄 🗄️

### やってたこと

Notionで日記を書いていた。データベース機能が強力だってんで、テンプレートをこねくり回して「完璧な日記システム」を作ろうとしていた。

🤦 システム作りに3日かけて、日記は3日で飽きた。本末転倒とはこのことだ。

### 何がダメだったか

書くのはまあいい。問題は**まとめるとき**だ。

週報とか月報とか作りたいじゃないか。「今月何やったっけ」って振り返りたいじゃないか。

💸 でもNotion AIは別料金。金がかかる。

結局どうしたか。Notionの内容をコピーして、ChatGPTに貼り付けて、「まとめて」って頼んでた。

📋 コピペ。コピペ。コピペ。

😂 アホか。何のためのデジタルツールだ。

---

## 2️⃣ Obsidian + Claude Desktop時代 - 光が見えた 💡

### Claude Desktopとの出会い

🎉 **Claude Desktop**が出た。

こいつがすごかった。ローカルのファイルを直接読める。つまりObsidianで書いたMarkdownファイルを、そのままClaudeに食わせられる。

✊ コピペ不要。革命だ。

### やってたこと

- 📝 Obsidianで日記を書く
- 🤖 Claude Desktopで「今週まとめて」って言う
- ✅ 終わり

これでいい。これでいいんだ。人類はここまで来た。

---

## 3️⃣ Logseq時代 - 寄り道 🚶

![Logseqの画面](https://user-images.githubusercontent.com/25513724/221387376-4dc419c2-0d0a-460c-a920-2d211e78b456.gif)

### なぜ乗り換えた

ObsidianからLogseqに移った理由は正直よく覚えてない。

「アウトライナー形式がいい」とか「デイリーノートがデフォルトで便利」とか、そんな理由だったと思う。

😅 要するに新しいもの好きなだけだ。

### 何がダメだったか

💀 問題は**同期**だ。

俺はWindowsとMacを行ったり来たりしている。ローカルファイルの同期がめんどくさい。Dropboxだのなんだの設定して、でもたまにコンフリクトして、ファイルが「競合コピー」になって、もう嫌だ。

🏚️ あと**分散問題**。Notionにも昔のデータがある。Obsidianにもある。Logseqにもある。どこに何があるかわからない。デジタルの汚部屋だ。

---

## 4️⃣ 現在 - 書かない日記術 🧠

### 💡 気づき

ある日、気づいた。

💻 **俺の活動、ほぼ全部GitHubに残ってる**。

コード書いたらコミット。ドキュメント書いてもコミット。設定いじってもコミット。

日記書かなくても、GitHubのコミット履歴見れば「今日何やったか」全部わかる。

🤷 じゃあ日記いらなくない？

### 🔧 現在のシステム

今はこうなっている。

#### GitHub to Calendar 🤖

⚙️ GitHub Actionsで毎朝7時に、前日のコミット内容をGoogleカレンダーに自動送信するやつを作った。

> [github-to-calendar](https://github.com/infoHiroki/github-to-calendar)

何もしなくても、朝起きたらカレンダーに昨日の活動が入ってる。

😎 俺は寝てただけ。システムが働いてくれた。これだ。これが正解だ。

#### Googleカレンダー 📅

「でも作業以外のことも記録したいじゃん」って思うだろ。

📱 Googleカレンダーの予定の説明欄に書けばいい。スマホからでも書ける。同期の心配もない。

#### Claude Code 🤖

振り返りたくなったらClaude Codeに聞く。

「先月何やった？」って聞けば、GitHubとカレンダーから引っ張ってきてまとめてくれる。

🧠 俺は質問しただけ。AIがまとめてくれた。

---

## 🤔 なぜこれで続いてるのか

### ✍️ 書いてないから

さっきも言ったけど、**書いてないから**続いてる。

🍺 毎日「今日の振り返り」とか書こうとするから続かないんだ。意志の力に頼るな。意志なんてものは二日酔いの朝に消え去る。

💪 システムに任せろ。俺は何もしない。GitHubが記録してくれる。それでいい。

### 🌍 どこでも使えるから

✈️ 旅行中でも問題ない。

作業するならGitHubにコミットすれば勝手に記録される。メモ残したいならGoogleカレンダーにスマホで書けばいい。

📱 **スマホ1台あれば完結**する。

昔は旅行中に日記が途切れて、そのまま習慣が崩壊するパターンが多かった。今はそれがない。
